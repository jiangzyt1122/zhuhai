# 快乐小鸡响应式布局修复说明

## 背景

快乐小鸡农场是单页小游戏，当前页面在小屏或横屏场景下使用固定最小高度，真实屏幕高度不足时会被裁切。由于页面禁止滚动，右下角“Lay Egg”按钮可能落到屏幕外，用户无法点击下蛋。

## 范围

- 调整 `happy_chicken_farm.html` 的视口高度策略，让游戏容器跟随当前屏幕高度。
- 优化矮屏下 HUD 和下蛋按钮的紧凑布局，保证核心操作按钮留在可见区域。
- 不调整游戏规则、计数逻辑、角色状态、孵化流程或持久化数据。

## 字段与数据

本次不新增业务字段，不新增持久化数据，不改变现有 DOM id 与游戏状态对象。

## 验收

- 在手机竖屏、小尺寸窗口和手机横屏高度不足的场景下，“Lay Egg”按钮可见且可点击。
- 页面仍保持全屏游戏体验，不出现需要滚动才能点击按钮的情况。
- 调整窗口大小后，游戏场景会继续按实际视口重新计算布局。
- 页面内脚本语法校验通过。
